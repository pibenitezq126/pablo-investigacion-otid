## Contenido tratado

### 1. Nueva asignación: Calculadora generada con IA

Se asignó como primera tarea práctica del grupo:

> Generar una aplicación (calculadora, de simple a sofisticada) usando un modelo de IA para generación de código, y mostrar en vivo el proceso completo de cómo se construyó.

**Puntos clave de la consigna:**
- El entorno y el lenguaje de programación son libres — cada integrante puede usar el que prefiera.
- Lo que se evalúa es el *proceso* de generación asistida por IA, no la herramienta específica.
- Habrá una segunda fase de análisis grupal sobre el ejercicio.

**Revisión ampliada (segunda entrega):**
Además de mostrar el código generado, se debe documentar y exponer:
- El entorno de trabajo completo
- El flujo del código de principio a fin
- Control de revisiones con el cliente (en este caso, el decano actuando como "cliente")
- Proceso de despliegue (tienda de apps o página web)
- Ubicación del código (repositorio)
- Manejo de solicitudes de cambio del cliente
- Documentación del proyecto
- Corrección de errores generados por la IA

> *Detalle del seguimiento de esta tarea en:* `otid-asignaciones/01-calculadora-ia/BITACORA.md`
